Envolée (ou Flipfly) est une application web interactive qui replace dans votre main l'avion-jouet de votre enfance.

Explorez en regardant autour de vous et réalisez les figures indiquées en utilisant votre appareil mobile.

Développée avec A-Frame, Fmod Studio et beaucoup d'enthousiasme.

Pour profiter de l'expérience, veuillez verrouiller votre téléphone en mode portrait.
